# 🤖 AI ChatBot App

A modern AI-powered chatbot application built with **Flutter**, featuring a beautiful UI, smooth animations, image support, and state management using **BLoC**.

---

## 📱 Features

* 💬 AI-powered chat experience
* 🎨 Modern and responsive UI
* ✨ Smooth animations
* 🖼️ Image picker support
* 🌊 Wave animations
* 🔄 State management with BLoC
* 💾 Local data persistence using Shared Preferences
* 🌐 API integration with Dio
* 🔤 Custom fonts (Poppins & Cairo)
* 📲 Android and iOS support

---

## 🛠 Tech Stack

* **Flutter**
* **Dart**
* **BLoC (flutter_bloc)**
* **Dio**
* **Shared Preferences**

## 🚀 Getting Started

### Prerequisites

* Flutter SDK >= 3.5.3
* Dart SDK >= 3.5.3

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ai_app.git
```

Navigate to the project directory:

```bash
cd ai_app
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

Run on Chrome:

```bash
flutter run -d chrome
```
